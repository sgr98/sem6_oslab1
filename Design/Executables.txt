The project has below executables( binaryexecutabels/ .out files )
There are 4 user groups in total
	1) student
	2) faculty
	3) admin
	4) hod

1) main			//this is the one every user executes to start using the application.
	executable by: anyone

2) student		//this is run if we identify the user to belong to student group
	executable by: student group
	
3) faculty		//this is run if we identify the user to belong to instructor group.
	executable by: faculty group

4) admin		//this is run if we identify the user to belong to admin group.
	executable by: admin group

5) hod			//this is run if we identify the user to belong to hod group.
	executable by: hod group
